# Tic Tac Toe (Python)

A simple command-line Tic Tac Toe game written in Python.
Two players (X and O) take turns playing on a 3×3 board until one player wins or the game ends in a draw.

## Features

Two-player game (X vs O)

Randomly selects the starting player

Validates moves to prevent overwriting cells

Detects:

Winning conditions

Draw situations

Clean, readable console board display

Object-Oriented design using a Tictactoe class

## How the Game Works

The board uses positions 1 to 9:

1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9


Players enter a number (1–9) to place their mark.

The game continues until:

A player gets 3 in a row, or

The board is full (Draw).

## Getting Started
### Prerequisites
Python 3.x installed on your system

### Run the Game

```
python tictactoe.py
```
